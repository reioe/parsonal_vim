## Diff

``` vim:vimcommand
:Gwrite
```

## 現在開いているソースの直前のコミット時のソースを表示

``` vim:vimcommand
:Gread
``` 

これはfugitive固有の機能っぽい。なにも考えずに実行すると問答無用で直前コミット状態に戻される。差分表示とかではない…
引数使うとなんかもっと有効にできそうな気もしたので、ファイル名とか付けてみたけど駄目。
調べればなんか出てくるだろうけど、
今のところは「元に戻す」だと思って使おう…
